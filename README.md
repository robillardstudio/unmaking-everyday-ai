# unmaking-everyday-ai data

Unmaking the Everyday AI: A Design Inquiry into Research Tools and AI Nudge. Text Mining Method

Borami Kang, Gaetan Robillard, Yvette Shen, Fabienne Munch

This repertory gives access to 5 csv file. Each lists the top keyword found in white papers or online pages published by platforms on responsible AI.
Two different types of parameters are given: string (word) and value (word frequency).

Platforms:
- Google
- O'Reilly Media
- Miro
- Notion
- Zoom
